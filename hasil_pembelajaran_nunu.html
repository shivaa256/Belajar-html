Ini salah satu cara biar bisa simpen gambar di web

Cara bikin foto di web itu lumayan "gesek kepala" ya ges ya. Di dalam data kamu harus ketik beberapa kode kayak img, src, alt, dan width.
Kita mulai dari img. Img singkatan dari image (gambar), cara penulisannya adalah <img>.
Lalu setelah img akan ada yang namanya src (source). Untuk apa itu src? Intinya src itu tempat kita masukin link gambar.
Selanjutnya adalah alt. Alt adalah teks alternatif kalau gambar gagal dimuat. Misalnya alt="kucing hitam".
Dan terakhir width. Kegunaannya untuk mengatur ukuran gambar (dalam pixel).

Cara bikin list di web

Caranya kamu tinggal pake kode <ol> atau <ul>.
Kita bahas dari ol dulu. Ol itu fungsi nya buat nambah list tapi pake nomor, contohnya:

  1. Suka makan
  2. Suka mandi

Untuk ul, itu hampir sama, tapi pake titik (bulat):

  • Suka makan
  • Suka mandi
  • Suka Shiva

Hal paling dasar dari coding adalah mengenali beberapa contoh yang ingin kamu kuasai, contohnya HTML dan Python

Beda nama, beda penggunaan ya bos.
Kita bahas beberapa kode yang dipakai di Python. Contohnya: input, print, if, elif, dan else.

Input: biasanya dipakai buat nanya nama.
Contoh: nama = input("Halo, boleh aku tahu siapa nama kamu?")

Print: untuk menampilkan jawaban. Misal: print("Halo Shiva, gimana kabarmu hari ini?")

If: untuk kondisi pertama. Misal: mood = input("Gimana kabar kamu hari ini Shiva?")
if "senang" in mood or "bahagia" in mood:
    print("Wah seneng dengernya!")

Elif: buat pilihan lain. Misal: elif "sedih" in mood or "nangis" in mood:
    print("Wah aku juga sedih denger nya. Kalo mau cerita aku dengerin kok.")

Else: buat jawaban lain kalau inputnya gak sesuai kondisi di atas.
Contoh: print("Wah sayangnya dataku belum punya fitur emosi itu. Tapi kamu bisa cerita kapan aja.")
